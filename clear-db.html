Limpiar Base de Datos KimiPOS

Este script eliminará la base de datos actual para forzar la actualización a la nueva versión que incluye la tabla de "Cuentas por Nombre".

Después de limpiar la base de datos:

  1. Ve a la aplicación KimiPOS
  2. Selecciona un salón
  3. Deberías ver la pestaña "Cuentas por Nombre" junto a "Mesas"
  4. Haz clic en "Cuentas por Nombre" para acceder a la nueva funcionalidad
